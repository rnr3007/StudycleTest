# StudcyleTest
Repository untuk memenuhi persyaratan Program Magang Bersertifikat di Studycle

## Penjelasan
* Bahasa yang digunakan adalah Python versi 3.0
* Sorting yang digunakan merupakan algoritma bubble sort
* Parameter dari setiap fungsi diberi nama `Data` untuk mencegah kebingungan
* Jika memasukkan input selain tipe data `int` ketika diminta, maka program akan terus meminta input bertipe `int` kecuali dengan memberhentikan paksa program
* Jangan lupa tekan `Enter` jika selesai menginputkan keyboard

## Cara Menggunakan
1. Pastikan Python 3.x telah terpasang
2. Download file `Studycle Assessment.exe` dan jalankan programnya
3. Data yang pertama kali diinputkan merupakan panjang data yang diinginkan
4. Data yang kedua dan seterusnya merupakan data yang akan diproses

## Contoh Penggunaan
Berikut ini ialah contoh penggunaan untuk memperjelas cara menggunakan
### Input
1. Masukkan nilai `5` ketika pertama kali muncul `Masukkan angka:` sebagai panjang data yang diinginkan
2. Selanjutnya, masukkan nilai `1`, `5`, `7`, `2`, `3` sebagai data yang akan diproses
